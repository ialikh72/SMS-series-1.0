# Shop Management System (SMS)

A simple **console-based Shop Management System** written in C#.  
It allows a shop owner to manage products, sell them, generate bills, and track total sales.

---

## Features

1. **Add Product**  
   - Enter product name, quantity, and price.  
   - Supports up to 100 products.  

2. **View Products**  
   - Display all products with ID, name, quantity, and price.  

3. **Sell Product**  
   - Sell a product by entering its ID and quantity.  
   - Shows the bill for the sold products.  

4. **Generate Bill**  
   - Generate a detailed bill for the last sold product.  
   - Adds the total to overall sales.  

5. **View Total Sales**  
   - Displays the total sales of the shop.  

6. **Exit**  
   - Close the program with a goodbye message.
